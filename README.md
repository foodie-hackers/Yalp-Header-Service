# Project Name
Yalp

# Project Description
Header microservice of a restaurant reservations application.

Full stack app built using Node, Express, React, MySQL. Deployed with Elastic Beanstalk, Amazon RDS.

https://www.youtube.com/watch?v=ZAtEe7uGSx8
